# 한 줄

```python
a = input()
print(a, type(a))
```

# 정수 입력

```python
 a = int(input())
 print(a, type(a))
```

# 한 줄에 여러개의 숫자

```python
 a, b = map(int, input().split(' '))
 print(a, b)
```

# 몫만 받기

```python
a = 100
b = 3
print(a / b)
print(a // b)
print(a % b)
```

# 문자 추가

```python
s = "hello"
print(s + "world!")
```
